**ChronoMaster** is a browser-based analog clock game designed to test and improve your time-reading skills. Players are challenged to read 10 random times displayed on an analog clock as quickly as possible.

1. **Start**: Click the "Start" button to begin the 3-second countdown.

3. **Type**: Enter the time in the input box in `HH:MM` format (e.g., `3:15`, `12:30`).
4. **Submit**: Press `Enter` or click "Submit".
   - If correct, you automatically advance to the next round.
   - If incorrect, try again!
5. **Finish**: Complete 10 rounds to see your final time and leaderboard position.
